⚙️ İlk Kurulum

SoundMatch için Spotify Developer hesabı gerekli. Aşağıdaki adımları takip et:

  1. developer.spotify.com/dashboard → Create App
  2. App Name: SoundMatch
  3. Redirect URI olarak şunu ekle: yükleniyor...
  4. APIs: Web API → Save
  5. Settings → Client ID ve Client Secret'ı kopyala
